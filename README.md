### Setup To Run Code
1. Create and activate a virtual environment(venv:
   ```bash
   python -m venv .venv
   
   .venv\Scripts\activate  # Windows
   source .venv/bin/activate  # Mac/Linux

2. pip install -r requirements.txt
3. Create a .env file and type in the correct DB credentials
4. Go to Backend Folder
5. npm install
6. Run node index.js
7. Run Webscraper by going to test_webscrape.http file and running first route
